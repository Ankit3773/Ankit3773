### 🚀 Featured Project

**🚌 Narayan Travel — Production Bus-Booking REST API**

A real-world booking backend built end-to-end and deployed live.

- **Java 17 + Spring Boot** with a layered architecture (controllers → services → repositories)
- **Spring Security** with JWT + Google OAuth, role-based access control (RBAC)
- **PostgreSQL** with pessimistic locking + a partial unique index to prevent double-booking
- **3-stage payment lifecycle** and **35+ REST endpoints** with integration tests
- **Deployed on AWS EC2** — Nginx reverse proxy, systemd service, Let's Encrypt HTTPS

🌐 **Live:** [narayantravels.duckdns.org](https://narayantravels.duckdns.org)
